## About Me

> *"Merging high-performance computing and AI to build intelligent systems ‚Äî one algorithm at a time."*

I'm a Computer Science student at Vellore Institute of Technology, Chennai with a passion for high-performance computing, machine learning, and cybersecurity. I specialize in developing acceleration techniques for computationally intensive tasks and building secure communication systems with a focus on efficiency.

<details>
<summary><b> More About My Journey</b></summary>
<br>
My work combines cutting-edge neural architectures with high-performance computing to create more efficient, secure AI systems. Beyond my core research, I'm expanding my technical skills through multiple projects involving real-time data processing, cryptographic systems, and financial forecasting.

Currently, I'm focusing on:
- Real-time sentiment analysis for financial applications using CUDA acceleration
- Building modular parallelized secure messaging architectures
- Exploring computational neuroscience through my GreyLattice project
</details>
